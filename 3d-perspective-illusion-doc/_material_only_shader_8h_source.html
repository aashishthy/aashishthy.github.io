50  // Both lights and materials have three color components - ambient, specular, diffuse
51  // "Ambient" light/color refers to what color a surface has when it is not exposed directly
52  // to light from the object. Think of an overcast day - everything in this case is lit by
53  // ambient lighting alone. Things in shadow are also lit by ambient lights alone.
54  // "Diffuse" light/color refers to the color when a light hits a surface, illuminating it.
55  // Most full-on light is visible as diffuse light.
56  // "Specular" light/color refers to the shiny spots on things like water and metallic surfaces
57  // When you look across a lake in the sunset, there's super bright white spots that appear
58  // where the reflection of the sun is. Those are lit primarily by specular light.
59  // For further information, read up on the Phong lighting model.
60  struct Material
61  {
62  Material(const Color& specular, const Color& diffuse, const Color& ambient)
63  : Specular(specular), Diffuse(diffuse), Ambient(ambient)
64  {}
